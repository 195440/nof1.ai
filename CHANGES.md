# Alpha-Beta 策略紧急修复

## 核心问题
- 胜率42.9%，盈亏比0.32:1，总亏损-11.99U
- AI开仓质量差：给ETH打88分，结果亏钱
- 评分门槛72分太容易达到

## 核心修复
1. **开仓门槛大幅提高**
   - Trend: 72→88分（+16分）
   - Range: 82→92分（+10分）
   - Chaos: 88→95分（+7分）

2. **7项严格检查**
   - 预期ROI≥5%
   - 多时间框架完全一致
   - 成交量≥1.2倍
   - 币种/方向失败保护
   
3. **移除DOGE**（单笔巨亏-8.96U）

4. **降低杠杆**（上限20→10倍）

5. **收紧止损**（-25/-20/-15 → -15/-12/-10）

6. **延长周期**（20→30分钟）

7. **添加资产曲线**（AI看到净值变化）

## 预期效果
- 开仓频率：↓70%（只做最确定的）
- 胜率：42.9% → 70%+
- 盈亏比：0.32:1 → 1.4:1+
- 总盈亏：-11.99U → +2~5U
